🪨 Last Translation Benchmark

A large-scale effort to collect difficult-to-translate inputs

This platform gathers inputs (text, video, audio, images, documents) that are challenging for machine translation systems. Contributors submit inputs with multiple MT outputs and a verification rule. Once approved, contributors have the option to be part of the resulting publication.

To access the platform, use the magic link you received. If you don't have one, contact the project administrators.
